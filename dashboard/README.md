# Dashboard Files

This folder contains CSV files designed for Tableau, Power BI, Excel, or Google Sheets.

| File | Purpose |
| --- | --- |
| cleaned_customer_churn.csv | Customer-level cleaned dataset with churn and revenue loss fields. |
| churn_summary.csv | Churn rate by contract type and payment method. |
| revenue_loss_summary.csv | Monthly and annual revenue loss by contract type and tenure group. |
| segment_analysis.csv | Segment-level churn and revenue metrics by age group, streaming plan, and tenure group. |

Suggested dashboard pages:

1. Executive KPI Overview
2. Churn Drivers by Contract and Payment Method
3. Revenue Loss Analysis
4. Customer Segment Deep Dive
5. Churn Reason and Retention Action Tracker
